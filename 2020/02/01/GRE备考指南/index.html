GRE备考指南

  • 背单词
    • 使用 考满分GRE3000词 这个软件的词频乱序新版进行背诵
    • 一天背200-300个单词
    • 会读一遍其中的近义词 和 反义词 相关 等词，这个跟GRE考同义词的思路相关，做题可能用得上
    • 一定要使用下面的遗忘曲线进行今日复习
      • 我会优先复习再学习新词
      • 保证所有词都得到充分的复习
  • Verbal
    • 在所有单词都背过一遍之后
    • 使用《张巍的GRE填空机经1250题.pdf》开始刷题
      • 倒着往前刷，我和小李都在考试的至少两个verbal里遇到原题
      • 甚至有一个有大量原题，我怀疑那个就是加试
    • 学有余力还可以再背诵一下等价词之类的
  • 阅读
    • 阅读也是从备考初期开始使用 杨鹏长难句 按照他的方法每天练习5句-10句
      • 预计消耗1个小时
    • 到后期开始刷题
      • 一定要按照他的方法不能回看
      • 且保证一边看懂
      • 我后来在考场就是不回看，一边看懂所有的意思
        • 虽然速度没有很快，但是选的题大部分都能选上
  • 作文
    • issue
      • issue 我后期主要使用的六分博客
        • 这个在我的issue练习笔记里的第六章节
          • 这是个计算机专业的博主，对每一道issue题都有练习，语言是那种平实的论证充分的
          • 我觉得语言没那么华丽，主要以例子取胜其实很适合我这样的语言不地道的人学习
        • 北美范文我前期在用，感觉还是有点难以模仿
          • 有些段落太高级了，自己写肯定写不出，练起来也就很无力
    • argument
      • argument我主要参考了《 GRE写作高频题目及考点精析.pdf》 这本书
      • 把6个direction各挑了一个例题，做了其中中文翻译成英文，最后和它的英文进行对比，看自己哪里翻译得不够好这样一个操作
      • 在这个中英翻译转换的过程里，同时还仔细留意了很多argument特有的过渡词（真的非常套路）
      • 我最后考试的时候也基本就是照着这个套路模版在写，速度较快
      • 可以参考我的《 Argument 4个类别练习》 这个md
    • 提纲
      • 因为GRE的作文题目是固定的188（？）个，它有放在GRE的官网上
        • 所以就可以根据高频题目练习
      • 对于argument来说，学有余力的话可以把每个题目的提纲，哪里有问题，哪里逻辑不完善等自己画一个思维导图，考试的时候就不用那么仔细读题目了
      • 而对于issue来说，就需要充分准备其中的高频题目，进行相应的练习（说不定可以遇到原题，我没遇到，但是我室友似乎写到的是她曾经写过的issue题）
